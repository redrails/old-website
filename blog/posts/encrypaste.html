EncryPaste

EncryPaste is a PHP project that basically does what it sounds like it does. It Encrypts some data that a user Pastes. It's an encryption pastebin which uses PHP-PDO to store the encrypted data and all encryption and decryption is done within PHP. No vulnerable information is stored.

The encryption is implemented using AES-256 and then wrapped around base_64 for a viewable format. The web-layout of the app is pretty simple, but the web-view is just a wrapper to show implementation. I've stopped working on this now but I always come back to add code in random things I've developed.
