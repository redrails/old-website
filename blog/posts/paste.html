Paste

Paste is my implementation of a pastebin. I first made it as a young kid when I as learning PHP and now I am using it as a base and inspiration to learn about Android Development. Paste has evolved greatly in the past and has now very simple, but useful features such as fast pasting, Markdown parsing and syntax highlighting.

The Android app makes use of Paste by using ASyncTask to achieve posting and getting requests from the server. Both the server and client go hand-in-hand and work to filter content and display it to the user. The work on Paste is on-going and is the smaller project I am undertaking.

If you would also like to develop on Paste then checkout this page!
