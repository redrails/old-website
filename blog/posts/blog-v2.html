Blog v2

I got really bored of Wordpress and the bloated style that it has. As a cleanfreak (only for code) myself, I like to have small software to work with, certainly not something that imposes a great load of stress both for the client side and server side of things. So this is my new blog, simple, clean and fast.

If you want to read some of my old posts you can find them here. But I don't intend on posting a wide range of topics anymore. Instead, I plan on posting more educational content, mostly about programming, code and other techy stuff, seeing as that's what I'm mostly about. An update for now, I'm working on many new projects, and Paste is one of the biggest ones. This development is ongoing and open-source, I'm mostly looking for feedback and with your help I may be able to make this an official platform!

Expect a lot more posts from now on! I'm also considering changing the layout of this website, not quite sure if I should. I quite like this condensed layout.
